EX-4: USING THE SIMPLE FILTERS VERIFY: pr, head, tail, cut, paste, ml, sort, grep, egrep, fgrep, write, and wall



AIM: To use the simple filters verify pr, head, tail, cut, paste, ml, sort, grep, egrep, fgrep, write, and wall.

PROCEDURE:

COMMAND: pr - convert text files for printing.

Syntax: pr [option] [filename]

OPTIONS MEANING
-d Double space the output
-D format Use format for the header date
-h header Use a centered 'header' instead of filename in page header
-l PAGE-LENGTH Sets the pagelength to PAGE-LENGTH lines
-O MARGIN Offset each line with MARGIN spaces
-W PAGE-WIDTH Set page width to PAGE-WIDTH characters

COMMAND:head- output the first part of file.

Syntax:head[option][filename]

Example:head-18 filename
Display the first 18 lines of the file called filename

COMMAND:tail- Displays lst 10(by default) lines of a file.

Syntax:tail[option][filename]

Example:tail-50 file.txt
Display the last 10 lines of the file.txt


COMMAND:cut- Remove section form each line of files.

Syntax:cut[option][filename]

OPTIONS MEANING
-b Bytes
-c characters

Example:cat a
This is linux
OK

cut-c2 a
h
k

cut-c2-3a
his
k


COMMAND:paste- Joins corresponding lines from files.

Syntax:paste[option][directory name]

OPTIONS MEANING
-d=LIST Reuse characters from the list insted of tab for the delimiters

Example:cat p1
1
one
ONE

cat p2
2
two
TWO
extra

cat p3
3
three
Three

cat p4
4
four
FOUR

paste -d="+_+" p3 p2 p1 p4
3+2-1=4
three+two-one=four
THREE+TWO-ONE=FOUR
+extra-=


COMMAND:nl- Number lines of file.

Syntax:nl[option][filename]

OPTIONS MEANING
-b STYLE Use style for numbering body lines style is one of the following types.
1) a-number all lines
2) t-number only non-empty lines
3) n-number no lines
pBRE-number only lines that contain a match for the basic regular expression

Example:nl -b t a
1 this is linux
2 OK
3 fine
4 end


COMMAND:sort- sort lines of text files.

Syntax:sort[option][filename]

Example:sort a
Alpha
Beta
Gamma
Omega

sort -r a
Omega
Gamma
Beta
Alpha

COMMAND:grep- Acronym for Global Regular Expression.

egrep-Extended for Global Regular Expression Print.

fgrep-Fixed-String Global Regular Expression Print.

Syntax:(e/f) grep[option]reg.exp[filename]

Example:
grep-n "os" filename:Display the linenumbers containing os.
grep-e "linux" -e "os" filename:Search for more than one expression
egrep +S filename:The characters S matches one or more times.
egrep 'is|or|go' filename:Lines whixh contain is|or|go
fgrep "." filename:Do not expand.(dot), instead Searchs the line having .(dot) literally.


COMMAND:write- The write utility sends a message to another user who is logged in.When you and another user use write to sedn message to each other,you establish two-way commnication.Initially a write command displays a banner on the other user's terminal,saying that yoy are about to send a message.

Syntax:write

Example:write max
Hi,Max,are you there?


COMMAND:wall- Send a message to everybody's terminal.

Syntax:wall[message][username]
The length of the message is limited to 20 lines
