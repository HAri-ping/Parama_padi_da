PART-A LINUX COMMANDS



EX-3: Use the General purpose commands:wc,cal,date,who,tty,ln


AIM:- To use the General purpose commands:wc,cal,date,who,tty,ln.

PROCEDURE:-

COMMAND:wc-Print the number of lines, words, and bytes in file.

Syntax:wc[option][file]

OPTIONS MEANING
-c bytes
-l lines
-c words

Example:wc-l filename
GIves the number of lines

wc-w filename
Gives the number of wordswc-c filename
Gives the number of characters


COMMAND:cal- the cal command display a simple calender.

Syntax:cal[option]
It display the current month,with weeks starting on sunday

Example:cal

COMMAND:date- display the system date and time.

Syntax:date

Example:date


COMMAND:who-print all username currently logged in.

Syntax:who
who i am
Reports the details about the command user

whoami
Reports the useranme of the command user.

COMMAND:tty- print the filename of the terminal connected to standard input.

Syntax:tty

Example:tty
(output)/dev/pts/o


COMMAND:ln- Makes links between files.

Syntax:ln target linkname
This will create a likn to 'target' with the name'linkname'

Example:ln file1 link1
