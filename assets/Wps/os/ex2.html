EX-2: USAGE OF FILE MANAGEMENT COMMANDS: cat, chmod, cp, mv, rm, more, file commands


AIM:- To use file management commands cat, chmod, cp, mv, rm, more, file commands.

PROCEDURE:-

COMMAND: cat - Concatenate files and display on the standard output.

Syntax: cat [option] [filename]

EXAMPLE MEANING
cat f1 (or) cat < f1 will display the content of the file f1
cat > f1
This is the content
Ctrl+D		 will create a file with the name f1
cat >> f1
Append this data
Ctrl+D

cat f1
This is the content
Append this data		 This command appends data to the already existing file f1
cat f1 f2 > f3 writes the content of f1 and f2 to f3

COMMAND: chmod - Used to change file access permission.

Syntax: chmod

Mode can be in symbolic representation or an octal number.

SYMBOLS MEANING SYMBOLS MEANING SYMBOLS (permission) MEANING
Oothers+add permissionrread
uowner-remove permissionwwrite
ggroup=assign permissionxexecute
aall

Syntax for symbolic values: chmod [u/g/o/a][+/-/=][permission]

Example: chmod u=rwx, g=r file1 (use ls -l filename to find permission level)

Syntax for octal values: chmod n1 n2 n3
n1 - octal value for owner
n2 - octal value for group
n3 - octal value for others

Octal values: r = 4, w = 2, x = 1, total = r + w + x

Example: chmod 777 f1

COMMAND: cp - Copy files and directories.

Syntax: cp [option]

Example: cp file1 file2 (copies file1 into file2)


COMMAND: rm - Remove files.

Syntax: rm [option]

Example: rm file1 (Removes file1)


COMMAND: mv - Move or rename files or directories.

Syntax: mv [option]

Example: mv filename1 filename2


COMMAND: more - Display text one screen at a time.

Syntax: more [option]

Example: more filename


COMMAND: file - Determine the file type.

Syntax: file [option]

Example: file one
one: ASCII text
