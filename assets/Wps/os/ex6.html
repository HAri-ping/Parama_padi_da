EX-6: Write a shell script to find factorial of the given number



AIM:- To write a shell script to find factorial of the given number.


PROGRAM-

#!/bin/bash
echo “Enter the Number”
read num
fact=1
for((i=2;i<=num;i++))
do
fact=$((fact*i))
done
echo $fact


OUTPUT:-



RESULT:-

Thus the shell script to find factorial of the given number is done and executed successfully.
