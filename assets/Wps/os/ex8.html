EX-8: WRITE A SHELL SCRIPT USING COMMAND LINE ARGUMENTS AND REPORTS ON WHETHER IT IS A DIRECTORY, FILE, OR SOMETHING ELSE



AIM:- To write a shell script using command line arguments and reports on whether it is a directory,file or something else.


PROGRAM-

#!/bin/bash
echo “Enter the filename”
read file
if [-f $file]
then
echo “File” elif [-d $file]
then echo “Directory”
else echo “something else”
fi


OUTPUT:-



RESULT:-

Thus the shell script to using command line arguments and reports on whether it is a directory,file or something else is done and executed successfully.
